# Kotlin-OOP

Share OOP document and demo.

## Overview
This repository contains documentation and demo code illustrating object-oriented programming (OOP) concepts using Kotlin.

## Features
- Example code showing OOP principles (classes, inheritance, interfaces, encapsulation, polymorphism)
- Demo projects and small exercises
- Notes and explanations for learning and teaching OOP in Kotlin

## Prerequisites
- JDK 8 or newer
- Kotlin toolchain (kotlinc) or IntelliJ IDEA
- Gradle or another build tool if a demo uses a build script

## Quick start
1. Clone the repository:
   git clone https://github.com/SarahLiMengqing/Kotlin-OOP.git

2. Open the project in IntelliJ IDEA (recommended) or import the Gradle project if present.

3. Build and run demos:
   - If the demo uses Gradle, run: ./gradlew run (or use the Run configuration in IntelliJ)
   - Or compile and run Kotlin files with the Kotlin compiler:
     kotlinc src -include-runtime -d demo.jar
     java -jar demo.jar

Refer to each demo folder for specific instructions.

## Repository structure (example)
- docs/       — OOP notes and documents
- demo/       — Demo projects and runnable examples
- src/        — Source code for examples
- examples/   — Small, focused samples

(Actual folders may vary; check the repository tree for current layout.)
